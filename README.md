# Spotify Clone

A full Spotify clone built with React and the Spotify Web API.

## Features

- Spotify OAuth Login
- Browse playlists
- Play tracks
- Spotify Web Playback SDK
- Responsive layout

## Tech Stack

React  
Vite  
Spotify API  
Spotify Web Playback SDK

## Setup

1. Clone repo

git clone https://github.com/yourusername/spotify-clone

2. Install

npm install

3. Create .env

VITE_SPOTIFY_CLIENT_ID=your_client_id  
VITE_REDIRECT_URI=http://localhost:5173

4. Run

npm run dev
